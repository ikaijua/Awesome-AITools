# Dexter 介绍

## 什么是 Dexter？

**Dexter** 是一个自主财务研究代理，旨在通过思考、计划和迭代学习进行深度分析。它的功能类似于“Claude Code”，但专门针对财务领域。它接收复杂的财务查询，将其分解为结构化的研究计划，并使用实时市场数据执行。

## 核心理念

### 专注于财务
与通用代理不同，Dexter 专门为财务分析量身定制，理解市场概念、财务报表和量化数据。

### 自主研究
Dexter 不仅仅是回答问题，它还在进行研究。它规划多个步骤，获取数据，验证其发现，并进行迭代，直到达成高置信度的结论。

### 透明的推理
凭借其“草稿本”日志系统和内置的反思功能，Dexter 提供了清晰的可见性，展示它是如何得出结论的，使其适用于需要高精度的数据密集型研究。

## 核心功能

### 智能任务规划
自动将复杂的财务问题分解为可操作的、分步骤的研究任务。

### 自主执行
独立选择并利用工具获取财务数据（如损益表、资产负债表、现金流量表）。

### 自我验证与反思
批判性地评估自己的发现并进行迭代，直到达成有数据支持的结论。

### 实时数据集成
连接到机构级财务数据集和网络搜索工具（Exa/Tavily）。

### 多渠道访问
支持通过 CLI 和 WhatsApp 网关进行交互。

### 评估套件
具有内置的测试框架，使用 LangSmith 根据财务数据集对代理的准确性进行评分。

## 快速入门

### 安装

```bash
# 克隆仓库
git clone https://github.com/virattt/dexter.git
cd dexter

# 使用 Bun 安装依赖
bun install
```

### 配置

您需要 LLM 提供商和数据源（例如 OpenAI、Financial Datasets API）的 API 密钥。按照仓库文档中的说明在 `.env` 文件中进行配置。

### 基本用法

```bash
# 运行 CLI
bun run start
```
