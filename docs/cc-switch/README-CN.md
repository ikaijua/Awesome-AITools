# CC Switch 介绍

## 什么是 CC Switch?

CC Switch 是一款跨平台桌面应用，旨在作为流行的 AI 命令行工具（包括 **Claude Code, Codex, Gemini CLI, OpenCode 和 OpenClaw**）的一站式管理器。它提供了一个可视化界面，可以同时管理多个工具的配置、API 供应商和 MCP 服务器。

## 核心功能

### 1. 多工具管理
- 五大 AI CLI 工具的统一界面。
- 无需手动编辑配置文件（JSON, TOML, .env）。

### 2. 供应商预设
- 内置 50 多个各种 API 供应商（如 AWS Bedrock, NVIDIA NIM, 社区中转等）的预设。
- 一键切换不同的供应商。

### 3. 统一的 MCP 和技能管理
- 单一面板管理 MCP 服务器和 AI 技能。
- 在所有支持的应用之间进行双向同步。

### 4. 系统托盘集成
- 直接从系统托盘快速切换供应商。
- 无需打开主程序即可完成常用操作。

### 5. 用量追踪
- 内置仪表盘，监控支出和请求日志。
- Token 使用趋势和图表。

### 6. 云端同步
- 在不同设备间同步配置数据。
- 支持 Dropbox, OneDrive, iCloud 或 WebDAV。

### 7. 会话管理
- 在所有管理的 AI 工具中浏览、搜索和恢复对话历史。

## 支持的工具

- **Claude Code**
- **Codex**
- **Gemini CLI**
- **OpenCode**
- **OpenClaw**

## 快速上手

### 安装

从 [GitHub Releases](https://github.com/farion1231/cc-switch/releases) 页面下载适用于您操作系统（macOS, Windows 或 Linux）的最新版本。

### 配置

1. 启动 CC Switch。
2. 前往 **Providers** 选项卡设置您的 API 密钥和预设。
3. 使用 **Tools** 选项卡启用并配置您喜欢的 AI CLI 工具。
4. 在相应的面板中管理 **MCP 服务器** 和 **技能**。
