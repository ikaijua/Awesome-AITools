# CC Switch Introduction

## What is CC Switch?

CC Switch is a cross-platform desktop application designed to serve as an all-in-one manager for popular AI-powered CLI tools, including **Claude Code, Codex, Gemini CLI, OpenCode, and OpenClaw**. It provides a visual interface to manage configurations, API providers, and MCP servers across multiple tools simultaneously.

## Key Features

### 1. Multi-Tool Management
- Unified interface for five major AI CLI tools.
- Eliminate the need for manual configuration file editing (JSON, TOML, .env).

### 2. Provider Presets
- Over 50+ built-in presets for various API providers (e.g., AWS Bedrock, NVIDIA NIM, community relays).
- One-click switching between providers.

### 3. Unified MCP & Skills
- Single panel to manage MCP servers and AI skills.
- Bidirectional synchronization across all supported apps.

### 4. System Tray Integration
- Quick-switch providers directly from the system tray.
- No need to open the main application for common tasks.

### 5. Usage Tracking
- Built-in dashboard to monitor spending and request logs.
- Token usage trends and charts.

### 6. Cloud Sync
- Synchronize configuration data across devices.
- Support for Dropbox, OneDrive, iCloud, or WebDAV.

### 7. Session Manager
- Browse, search, and restore conversation histories across all managed AI tools.

## Supported Tools

- **Claude Code**
- **Codex**
- **Gemini CLI**
- **OpenCode**
- **OpenClaw**

## Quick Start

### Installation

Download the latest release from the [GitHub Releases](https://github.com/farion1231/cc-switch/releases) page for your operating system (macOS, Windows, or Linux).

### Configuration

1. Launch CC Switch.
2. Go to the **Providers** tab to set up your API keys and presets.
3. Use the **Tools** tab to enable and configure your preferred AI CLI tools.
4. Manage **MCP Servers** and **Skills** from their respective panels.
